Sobre mim

Tenho 24 anos de idade, sou casado e moro em Pinhais, região metropolitana de Curitiba, Paraná.

Tive meu primeiro contato com desenvolvimento de software pela curiosidade de como os jogos são feitos, então estudei C# para criar jogos na Unity Engine. Depois disso, aprendi sobre desenvolvimento web com HTML, CSS e JavaScript.

Alguns anos depois comecei a trabalhar em uma startup de tecnologia no atendimento ao cliente, naturalmente tive um maior contato com o desenvolvimento e encontrei minha profissão.

Desde então venho desenvolvimento hard skills que envolvem, mas não se limitam, à Ruby On Rails, Node, React (Native e Next.js), Nest.js, Docker, PostgreSQL, MySQL, microsserviços entre outros.
